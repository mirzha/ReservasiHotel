# Tugas Besar Mata Kuliah Proyek
### Aplikasi ini adalah aplikasi untuk melakukan reservasi Hotel, disini terdapat 3 cara akses.
### http://localhost/tugaspryk/ --> untuk menampilkan halaman utama dari website
### http://localhost/tugaspryk/backend --> untuk menampilkan halaman dari admin
### http://localhost/tugaspryk/rating/ --> untuk menampilkan halaman pemberian rating oleh user

#### Khusus untuk backend
##### Database = gameshop
##### username = admin
##### password = admin
